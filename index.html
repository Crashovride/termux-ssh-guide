Connect to your Android device via SSH using Termux

Step-by-step instructions, tips, and tricks to master remote access from your phone.

Quick Setup

  1. Install Termux from F-Droid.
  2. Run: pkg install openssh
  3. Start SSH server: sshd
  4. Find IP: ip a
  5. Connect: ssh username@device-ip
