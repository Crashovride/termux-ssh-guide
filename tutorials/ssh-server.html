Running SSH Server in Termux

Step-by-Step Guide

  1. Install Termux from F-Droid.
  2. Update packages: 
    pkg update && pkg upgrade
  3. Install OpenSSH: 
    pkg install openssh
  4. Create a password for your Termux user: 
    passwd
  5. Start the SSH server: 
    sshd
  6. Check your local IP address: 
    ip a
  7. Now SSH into the device from another machine: 
    ssh u0_aXXX@
    Replace u0_aXXX with your actual Termux username.

Tip: Use termux-info to check system details.
