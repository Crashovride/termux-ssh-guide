Using Termux as an SSH Client

How to Connect to Remote Servers

  1. Install OpenSSH if not already installed: 
    pkg install openssh
  2. Connect to a remote server: 
    ssh username@remote-ip
  3. Accept the host key and enter your password when prompted.
  4. You now have terminal access to the remote machine.

Bonus: Use -p to specify a port (e.g. ssh -p 2222 user@ip).
